Customer Segmentation — ML Web App

This project uses unsupervised machine learning to segment retail customers based on purchase behavior and visualize them in a Streamlit web app.

Objective

• Analyze transactions
• Create RFM features
• Cluster similar customers
• Visualize segments

Algorithms

• K-Means
• DBSCAN
• Hierarchical Clustering

Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Streamlit

Run

Train model:

python train_model.py

Run app:

streamlit run app.py
Key Insight

High-value customers generate most revenue.
